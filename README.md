# tools.lborgav.com

A collection of small, useful web tools.

## Inspiration

This project is inspired by [Simon Willison's Tools](https://tools.simonwillison.net/) — a showcase of focused, single-purpose utilities that solve specific problems well.

## Available Tools

- **[dados-econ-br](./dados-econ-br)** — Brazilian economic indicators dashboard
- **[whatsapp-exporter-viewer](./whatsapp-exporter-viewer)** — WhatsApp chat export viewer
- **[json-viewer](./json-viewer)** — Beautify and view JSON with syntax highlighting

## Running Locally

Each tool is a standalone web application. See individual tool directories for setup instructions.
